# FruitFusion: A Visual Nutrition Dashboard

Overview

FruitFusion is a Power BI dashboard project that explores the nutritional and physical characteristics of fruits using a compact Kaggle dataset. The report was designed to compare fruits across measures like sugar content, fiber, water percentage, acidity, shelf life, and seasonality.

This project focuses on clean dashboard design, exploratory analysis, and visual storytelling.

Objectives

The goal of this project was to:

Build a visually engaging Power BI dashboard from a small dataset

Compare fruits by nutrition and composition

Explore relationships between sugar, acidity, water content, and shelf life

Practice dashboard layout, slicers, scatter plots, KPI cards, and multi-page storytelling

Dashboard Pages
Page 1: Nutrition Overview

This page provides a broad overview of fruit nutrition and seasonality.

Key visuals include:

KPI cards for fruit count, average weight, water content, and fiber

Donut chart showing fruit distribution by season

Scatter plot comparing fruit weight and water content, with bubble size representing sugar

Bar chart of top fruits by calories and sugar content

Bar chart of highest-fiber fruits

Page 2: Fruit Characteristics & Storage Insights

This page focuses on fruit behavior and composition.

Key visuals include:

KPI cards for average shelf life and taste profile count

Scatter plot comparing sugar content and acidity, with bubble size representing water percentage

Water content comparison

Additional visuals focused on fruit characteristics and storage-related insights

Key Insights

Some of the main observations from this project include:

Most fruits cluster around moderate acidity levels

Sugar content varies more widely than acidity

There does not appear to be a strong correlation between acidity and sugar content

Certain fruits stand out as high-fiber options

Seasonal distribution is uneven, with some seasons containing a larger share of fruits in the dataset
